## 🚀 Projetos em Destaque

### 🤖 [Agente Meteorológico — LLM com Validação e Fallback Inteligente](https://github.com/jvgomeswhatch) `2026`
> Agente de previsão do tempo confiável com mecanismos anti-alucinação, fallback via Regex e memória contextual.

**Desafio:** Modelos LLM locais apresentavam risco de alucinação ao consumir dados meteorológicos em tempo real.  
**Solução:** Agente em Python com Ollama + Llama 3.2, Function Calling, integração Open-Meteo, validação determinística e fallback inteligente. Interface web via Gradio.  
**Resultado:** Redução de inconsistências nas respostas do LLM e garantia de integridade entre resposta gerada e dados reais da API.

`Python` `Ollama` `LLM` `Function Calling` `Gradio` `Prompt Engineering` `Fallback Systems`

---

### 🏟️ [IA Scout — Plataforma de Scouting com LLM Local](https://github.com/jvgomeswhatch) `2026`
> Plataforma inteligente de análise de jogadores com IA, sem dependência de APIs pagas.

**Desafio:** Criar sistema de scouting completo com LLM local, limitado a infraestrutura com 16GB RAM.  
**Solução:** Backend FastAPI + Frontend React/TypeScript + LLMs via Ollama com pipeline de entity resolution, fuzzy matching, feature engineering e decision engine. Autenticação JWT e suíte completa de testes automatizados.  
**Resultado:** Plataforma com análises automatizadas de contratação, baixo custo operacional e fallback inteligente de modelos.

`Python` `FastAPI` `React` `PostgreSQL` `Ollama` `LLM` `JWT` `Docker` `AI Pipeline`

---

### ⚙️ [Event-Driven Microservices Platform](https://github.com/jvgomeswhatch) `2026`
> Plataforma distribuída de processamento de pedidos com mensageria, observabilidade e tolerância a falhas.

**Desafio:** Simular arquitetura distribuída AWS localmente para estudo avançado de sistemas orientados a eventos.  
**Solução:** 4 microserviços em Go integrados via SNS/SQS com LocalStack e Terraform, implementando idempotência, DLQ, correlation ID e observabilidade com Loki/Grafana.  
**Resultado:** Processamento assíncrono resiliente com rastreamento completo de requisições e infraestrutura totalmente automatizada.

`Go` `Microservices` `Terraform` `Docker` `SNS` `SQS` `DynamoDB` `Loki` `Grafana`

### 🌩️ [WeatherSOS — Plataforma Serverless para Desastres Naturais](https://github.com/jvgomeswhatch) `2025`
> Aplicação de suporte emergencial com alta disponibilidade e escalabilidade automática.

**Desafio:** Suportar picos críticos de acesso em cenários de desastres naturais com simplicidade operacional.
